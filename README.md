# 🍕 Zomato-Inspired Static Website – Deployed with GitHub Actions & Pages

This project is a **Zomato-style food delivery landing page**, built using `HTML`, `CSS`, and `JavaScript`, and deployed using **GitHub Actions** with **GitHub Pages**. It showcases how to automate the deployment of a static website with a modern DevOps workflow.

---

## ❓ The Problem

Manually deploying static websites every time you make a change is:
- ❌ Time-consuming  
- ❌ Error-prone  
- ❌ Not scalable  

Beginners also often struggle to understand how to integrate **CI/CD pipelines** and **automated deployments**.

---

## ✅ What This Project Solves

This project shows how to:

- 📦 **Host a static site for free** using GitHub Pages  
- ⚙️ **Automate deployment** using GitHub Actions  
- 👨‍💻 **Write clean, responsive websites** with HTML, CSS, JS  
- 🚀 Set up a real-world **CI/CD pipeline** that auto-deploys on every push to `master`

## 🚀 Implementation Steps

1. **Create project structure** with `index.html`, `style.css`, and `app.js`
2. **Initialize Git** and create a GitHub repository
3. **Push code to GitHub** on the `master` branch
4. **Create `.github/workflows/deploy.yml`** for GitHub Actions CI/CD pipeline
5. **Enable GitHub Pages** in repository settings (set source to GitHub Actions)
6. **Trigger deployment** automatically by pushing changes to `master`
7. **Access your live site** via the GitHub Pages URL
